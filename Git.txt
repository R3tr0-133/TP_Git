Git - Cours 1

1. Créer un repertoire sur GitHub		|	new repository
2. Créer un dossier 					|	mkdir nom_dossier
3. Créer un fichier text dans ce dossier 	|	nano nom_fichier
4. Ajouter une branche					|	
4. Initialiser le dossier 					|	git init 
5. Ajouter l'origine de repertoire 			|	git remote add origin https://github.com/utilisateur/...
6. Ajouter l'email de l'utilisateur 			|	git config user.email "exemplemail@gmail.com"
7. Ajouter le nom de l'utilisateur 			|	git config user.name "nom_utilisateur"
8. Ajouter le fichier au répertoire 		|	git add nom_fichier
9. Premier commit 					|	git commit -m "first commit"
10. Push 							|	git push -u origin "nom de la branche"
									|	Password = Token
